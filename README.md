## Cenário e Fonte de Dados
O dataset utilizado contém informações sobre os 100 melhores filmes segundo o IMDb e outras fontes.  
Cada registro possui atributos como título, ano, diretor, gênero, país, nota IMDb, bilheteria, entre outros.

A base de dados foi obtida de uma fonte pública (Top 100 Movies Dataset) e será usada para demonstrar
operações transacionais (INSERT, UPDATE, DELETE) em tabelas Delta e Iceberg com Apache Spark.
